# Swift Asset

[English](#english) | [Français](#francais)

---

<h2 id="english">English</h2>

Swift Asset is a premium, client-side web tool designed for optimizing images and converting SVGs directly into React components. It features a high-end "Laboratory" aesthetic with interactive, glassmorphic UI elements and guarantees maximum privacy by performing all processing locally within the browser.

### Features
- **Local Processing:** No server uploads. All image optimization and SVG parsing happen securely in your browser.
- **Image Matrix Module:** Quantize JPG/PNG formats with sub-pixel precision using native Web APIs.
- **SVG Precision Engine:** Clean and convert raw SVGs to clean, ready-to-use React JSX components.
- **Batch Processing:** Drag and drop multiple files for parallel optimization.
- **Action Log:** Persistent history of your processed assets during the current session.
- **Responsive Design:** Premium UI optimized for Desktop, Tablet, and Mobile.

### Stack
- React 19
- Vite 6
- Tailwind CSS v4
- Framer Motion
- Phosphor Icons

### Development Setup
To run the project locally:

```bash
# Install dependencies
npm install

# Start the development server
npm run dev
```

### Deployment (Vercel)
This project is optimized for deployment on Vercel:
1. Push this repository to GitHub.
2. Import the project in your Vercel dashboard.
3. The build command will automatically be detected as `npm run build` and the output directory as `dist`.

<h2 id="francais">Français</h2>

Swift Asset est un outil web premium, fonctionnant uniquement côté client, conçu pour optimiser les images et convertir les fichiers SVG directement en composants React. Il propose une esthétique "Laboratoire" haut de gamme avec des interfaces en glassmorphism et garantit une confidentialité totale du fait que tout le traitement est effectué localement dans le navigateur.

### Fonctionnalités
- **Traitement Local :** Aucun envoi sur un serveur. L'optimisation des images et le traitement SVG se font de manière sécurisée dans votre navigateur.
- **Module Matrice d'Images :** Quantifiez les formats JPG/PNG avec une précision au sous-pixel via les API Web natives.
- **Moteur de Précision SVG :** Nettoyez et convertissez les SVG bruts en composants React JSX propres et prêts à l'emploi.
- **Traitement par Lot :** Glissez et déposez plusieurs fichiers pour une optimisation en parallèle.
- **Journal d'Action :** Historique persistant de vos ressources traitées pendant la session active.
- **Design Réactif :** Interface utilisateur premium optimisée pour Ordinateur, Tablette et Mobile.

### Technologies
- React 19
- Vite 6
- Tailwind CSS v4
- Framer Motion
- Phosphor Icons
